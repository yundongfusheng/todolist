# 目标

在 About 页面底部新增一个绿色「返回首页」按钮，方便用户快速导航至首页。

# 需求

- About 页面底部需展示一个「返回首页」按钮
- 按钮颜色为绿色（使用系统主色或指定色值 #28a745）
- 点击按钮后跳转至网站首页（根路径 `/`）
- 按钮位于页面内容区底部，视觉上与页面其他内容保持合理间距
- 按钮文案为「返回首页」

# 验收标准

- Given 用户进入 About 页面，When 页面加载完成，Then 页面底部可见一个绿色「返回首页」按钮
- Given 用户看到「返回首页」按钮，When 用户点击该按钮，Then 页面跳转至网站首页（`/`）
- Given 按钮处于正常状态，When 视觉检查，Then 按钮背景色为绿色，文字清晰可读，与页面风格一致
- Given 用户使用移动端设备，When 查看 About 页面，Then 按钮在小屏幕上正常显示，不溢出或遮挡其他内容
